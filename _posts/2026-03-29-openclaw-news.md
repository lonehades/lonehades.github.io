## 🗞️ Morning Report

### 1. Anthropic Model Context Protocol (MCP)
- **요지:** AI가 외부 데이터·도구와 연결되도록 표준 프로토콜과 SDK를 공개했습니다.
- **왜 중요한가:** 에이전트가 실제 시스템과 양방향으로 연결되어 자율 실행·도구 호출을 수행하는 기반 레이어입니다.
- **링크:** [Anthropic MCP](https://www.anthropic.com/news/model-context-protocol)

### 2. Palo Alto Networks Prisma Browser for Agentic AI
- **요지:** 에이전트 워크플로우를 안전하게 지원하는 보안 브라우저 기능 확대를 발표했습니다.
- **왜 중요한가:** 브라우저 상 자율 에이전트의 작업·권한·보안 통제는 실전 도입의 핵심 제약을 푸는 축입니다.
- **링크:** [Prisma Browser 발표](https://www.prnewswire.com/news-releases/palo-alto-networks-unveils-the-industrys-most-secure-browser-built-for-agentic-ai-302722477.html)

### 3. AWS Nova Act 브라우저 에이전트 심층 공개
- **요지:** 브라우저 자동화 전용 모델과 오케스트레이션 구조를 공개하며 높은 신뢰도를 주장했습니다.
- **왜 중요한가:** 스크린샷→추론→행동으로 이어지는 인지-행동 루프를 모델 차원에서 밀어붙인 사례입니다.
- **링크:** [Nova Act 딥다이브](https://dev.to/mohammed_anes_6652d05cab7/amazon-nova-act-deep-dive-perceive-act-deploy-how-aws-built-a-90-reliable-browser-agent-4ao6)

### 4. Claude Computer Use 비즈니스 활용 사례
- **요지:** GUI 기반 반복 업무를 실제 비즈니스 유스케이스로 연결한 사례들을 정리했습니다.
- **왜 중요한가:** API가 없는 환경에서도 화면을 보고 조작하는 시각-행동 루프가 실전 자동화를 가능케 합니다.
- **링크:** [Business use cases](https://www.mindstudio.ai/blog/claude-code-computer-use-business-use-cases-2)
