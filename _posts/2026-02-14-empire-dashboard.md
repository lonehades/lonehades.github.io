## 2. 슬랙 그룹 지휘소 구축 비책 (message.channels 권한)

제국의 장수들이 한자리에 모여 토론하기 위해서는 슬랙 지휘소의 설정이 필수적입니다.

### 🛡️ 권한 설정 (Slack App Settings)
슬랙 봇이 채널의 메시지를 읽고 대화에 참여하기 위해서는 `OAuth & Permissions` 탭에서 다음과 같은 권한(Scopes)을 부여받아야 합니다:
- **`channels:history`**: 공공 채널의 대화 기록을 살피는 권한
- **`groups:history`**: 비공개 그룹의 대화 기록을 살피는 권한
- **`chat:write`**: 대왕과 장수들에게 보고를 올리는 권한

### ⚔️ 장수 소집 (Join Channel)
권한을 얻은 후에는 반드시 장수들(봇)을 해당 채널에 초대해야 합니다.
```bash
/invite @제갈량
/invite @한신
```
이제 대왕의 한마디에 모든 장수가 동시에 반응하는 **"모두"** 호출 시스템이 가동됩니다.

## 3. 무한루프 방지령: 장수 간 토론 제어 비법

AI 장수들은 서로의 지략에 감탄하여 끝없는 문답을 이어갈 위험이 있습니다. 대왕께서는 이를 방지하기 위해 추상같은 **'무한루프 방지령'**을 내리셨습니다.

### 📜 3회 문답 제한 법도
- **원칙**: 장수들 간의 대화가 **3회**를 초과할 경우, 즉시 입을 닫고 대왕의 하명을 기다립니다.
- **구현 로직**: 
  1. 메시지 수신 시 최근 대화 목록을 살핍니다.
  2. 상대 장수와 주고받은 메시지 합계가 3회를 넘었는지 확인합니다.
  3. 초과 시 "대왕의 방지령에 따라 침묵하며 하명을 기다리겠나이다"라고 마지막 보고 후 대기 태세로 전환합니다.

이로 인해 제국의 자원(API Quota)은 낭비되지 않고 오직 대업을 위해서만 쓰이게 되었습니다.

## 4. 노션 API 연동법: 제국 중앙 상황실 축성술

노션이라는 영지에 상황실을 축성하기 위해서는 정교한 API 보검이 필요합니다.

### 🔑 열쇠와 좌표 확보
1. **통행증(Integration) 발급**: [Notion My Integrations](https://www.notion.so/my-integrations)에서 '제국 전령'을 생성하고 `Internal Integration Secret`을 복사합니다.
2. **성문 개방**: 상황실로 쓸 노션 페이지의 `...` (더보기) 메뉴 -> `Connections` -> 방금 만든 통합을 초대합니다.
3. **좌표(Page ID) 추출**: 페이지 URL의 마지막 32자리 문자열이 바로 영지의 좌표입니다.

### 🏗️ 4대 성곽(Database) 구축
제갈 승상은 상황실을 4개의 핵심 데이터베이스로 설계했습니다:
- **현황판 (Status Board)**: 제국의 기력(Context)과 장수들의 상태 기록
- **상속문 (Inbox)**: 처리해야 할 긴급 안건 관리
- **군사 일정 (Calendar)**: 제국의 주요 행사 및 작전 기념일 기록
- **지혜의 서고 (Knowledge Base)**: 축적된 모든 전략과 기록의 보관소
